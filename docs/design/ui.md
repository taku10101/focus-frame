## 3. インタラクション仕様

### 3.1 マス開放アニメーション

ポモドーロ完了時、1マスが開放される瞬間の演出。**ユーザーの達成感を最大化する最重要インタラクション。**

```
[Phase 1: 予告] 300ms
  - グリッド全体が軽く pulse (scale 1.0 → 1.02 → 1.0)
  - 効果音: 短い上昇チャイム

[Phase 2: 選択] 500ms
  - ランダムなマスの上を光のカーソルが高速で移動
  - 3〜5マスを経由してターゲットに到達
  - easing: cubic-bezier(0.34, 1.56, 0.64, 1)

[Phase 3: 開放] 400ms
  - ターゲットマスが白くフラッシュ (opacity 0 → 1 → 0)
  - 同時にマスの色が unrevealed → 実際の色に transition
  - box-shadow で glow エフェクト (--shadow-cell)
  - transform: scale(1.3) → scale(1.0) with bounce

[Phase 4: 定着] 200ms
  - glow が fade out
  - 進捗カウンターがインクリメント (+1 アニメーション)
  - 触覚フィードバック (navigator.vibrate(50))
```

**CSS Animation 概要:**
```css
@keyframes cell-reveal {
  0%   { transform: scale(1); opacity: 0; }
  30%  { transform: scale(1.3); opacity: 1; background: white; }
  60%  { transform: scale(1.1); background: var(--actual-color); }
  100% { transform: scale(1); background: var(--actual-color); }
}
```

### 3.2 リビール完了演出

全マス開放時の特別演出。**コレクションにおける最大のご褒美。**

```
[Phase 1: 最後のマス開放] 400ms
  - 通常のマス開放アニメーション

[Phase 2: 波紋] 800ms
  - 最後のマスから波紋状に全マスが輝く
  - 各マスが順番に scale(1.05) → scale(1.0)
  - 遅延: 中心からの距離 × 50ms

[Phase 3: タイトルリビール] 1200ms
  - グリッド全体が上にスライド (translateY(-20px))
  - 下から作品タイトルが fade-in + slide-up
  - タイトルはタイプライター風に1文字ずつ表示
  - フォント: heading-xl
  - 作者名は 300ms 遅延で fade-in

[Phase 4: 祝福] 2000ms
  - 画面全体にパーティクル (紙吹雪) エフェクト
  - CSS only: 30個の小さな四角形が散る
  - 色は作品の主要色から抽出
  - 「🎉 完成おめでとう！」テキスト fade-in
  - 「コレクションに追加しました」toast 通知

[Phase 5: アクション] -
  - 「シェアする」「コレクションを見る」「次の絵を選ぶ」ボタン
  - staggered fade-in (各100ms遅延)
```

### 3.3 タイマー状態遷移

```
                    ┌─────────────┐
                    │   IDLE      │ ← 初期状態 / リセット後
                    │  "スタート"  │
                    └──────┬──────┘
                           │ tap start
                           ▼
                    ┌─────────────┐
              ┌────▶│  RUNNING    │ ← カウントダウン中
              │     │  "一時停止"  │    パルスアニメーション
              │     └──────┬──────┘    プログレスリング表示
              │            │
              │    tap      │ tap pause
              │    resume   │
              │            ▼
              │     ┌─────────────┐
              └─────│  PAUSED     │ ← 一時停止中
                    │  "再開"      │    リング一時停止
                    └─────────────┘    warning色

           time up │ (RUNNING → 0:00)
                   ▼
            ┌─────────────┐
            │ COMPLETED   │ ← マス開放アニメーション
            │ 🎉          │    通知 + バイブ
            └──────┬──────┘
                   │ animation end
                   ▼
            ┌─────────────┐
            │   BREAK     │ ← 5分休憩
            │  "スキップ"  │    break-accent色
            └──────┬──────┘
                   │ time up / skip
                   ▼
                 IDLE (次のセッション)

  ※ 長押し (500ms) でリセット確認ダイアログ
  ※ COMPLETED 時にすべてのマスが開放済み → リビール完了演出へ
```

### 3.4 画面遷移トランジション

| 遷移 | アニメーション | Duration | Easing |
|------|--------------|----------|--------|
| ホーム → 設定 | 左からスライドイン | 300ms | ease-out |
| ホーム → コレクション | 右からスライドイン | 300ms | ease-out |
| コレクション → 作品詳細 | 共有要素トランジション (グリッド画像が拡大) | 400ms | ease-in-out |
| ホーム → 絵の選択 | 下からスライドアップ (ボトムシート風) | 350ms | cubic-bezier(0.32, 0.72, 0, 1) |
| 戻る | 各遷移の逆方向 | 250ms | ease-in |

**共有要素トランジション (View Transitions API):**
- コレクション一覧のサムネイル → 作品詳細の拡大画像
- `view-transition-name: artwork-{id}` で紐付け

```
│   名画が見えてくる。        │
│                            │
│   [はじめる →]              │
│                            │
└────────────────────────────┘
  表示: fade-in 800ms

Step 2: コンセプト説明 (コーチマーク不要、1画面で完結)
┌────────────────────────────┐
│                            │
│  ① 🍅 25分集中する         │
│      ↓                     │
│  ② 🟩 マスが1つ開く        │
│      ↓                     │
│  ③ 🖼  名画が現れる        │
│      ↓                     │
│  ④ 🎉 コレクションに追加   │
│                            │
│   [最初の絵を選ぶ →]        │
│                            │
└────────────────────────────┘
  各ステップは staggered fade-in (200ms間隔)

Step 3: 絵の選択画面へ遷移
  → 通常の「絵の選択」画面を表示
  → 初回はツールチップで「まずは5×5がおすすめ！」を表示

Step 4: ホーム画面 (絵が選択された状態)
  → グリッドが全てグレーの状態で表示
  → スタートボタンに pulse アニメーション
  → ツールチップ: 「タップして集中を始めよう！」(3秒後消去)
```

### 設計方針
- **最短2タップでタイマー開始**できるようにする
- 説明は最小限。操作しながら理解させる
- 初回のみ。2回目以降は直接ホーム画面
- スキップ可能（ヘッダーに「スキップ」テキストリンク）

## 5. アクセシビリティ

### 5.1 色覚多様性対応

| 対応項目 | 方針 |
|---------|------|
| コントラスト比 | 全テキストで WCAG AA (4.5:1) 以上を保証 |
| 色だけに依存しない | マスの状態は色+パターン (未開放=斜線パターン) で区別 |
| タイマー状態 | 色+アイコン+テキストの3重表現（▶/⏸/✅） |
| 成功/エラー | 色+アイコンの併用 (✅ / ⚠️ / ❌) |
| フィルターチップ | 選択状態はfill色 + チェックマーク + border |

### 5.2 キーボード操作

| キー | アクション |
|------|----------|
| `Space` / `Enter` | タイマー開始/一時停止 |
| `R` | タイマーリセット（確認ダイアログ） |
| `Tab` | フォーカス移動 |
| `Escape` | モーダル閉じる / 前の画面に戻る |
| `1-5` | ホーム画面のナビゲーション |
| `Arrow Keys` | グリッドセル間のフォーカス移動 |

- **フォーカスリング**: `outline: 2px solid var(--accent-primary); outline-offset: 2px`
- **フォーカストラップ**: モーダル内でTabが循環

### 5.3 スクリーンリーダー対応

| 要素 | aria属性 |
|------|---------|
| タイマー | `role="timer"` `aria-live="polite"` `aria-label="残り25分00秒"` |
| グリッド | `role="grid"` 各セルに `role="gridcell"` |
| 未開放マス | `aria-label="未開放のマス (15/64)"` |
| 開放済みマス | `aria-label="開放済み: 暗い青色"` |
| 進捗 | `role="progressbar"` `aria-valuenow="15"` `aria-valuemax="64"` |
| フィルターチップ | `role="checkbox"` `aria-checked` |
| 画面遷移 | `aria-live="polite"` でページ変更を通知 |

### 5.4 モーション配慮

```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
  /* マス開放は色変更のみ（アニメーションなし） */
  /* パーティクルエフェクト無効 */
  /* 画面遷移はカット */
}
```
