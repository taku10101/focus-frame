# Webインターフェースガイドライン

Webインターフェースガイドラインへのファイルの準拠をレビューします。

## 動作の仕組み

1. 以下のソースURLから最新のガイドラインを取得する
2. 指定されたファイルを読み込む（指定がない場合はファイル/パターンをユーザーに確認する）
3. 取得したガイドラインの全ルールと照合する
4. 簡潔な `ファイル:行番号` 形式で結果を出力する

## ガイドラインのソース

レビューの前に最新のガイドラインを取得してください：

```
https://raw.githubusercontent.com/vercel-labs/web-interface-guidelines/main/command.md
```

WebFetchを使って最新のルールを取得します。取得したコンテンツには全ルールと出力フォーマットの指示が含まれています。

## 使い方

ファイルまたはパターンが引数として指定された場合：
1. 上記のソースURLからガイドラインを取得する
2. 指定されたファイルを読み込む
3. 取得したガイドラインの全ルールを適用する
4. ガイドラインで指定されたフォーマットで結果を出力する

ファイルが指定されていない場合は、レビュー対象のファイルをユーザーに確認してください。
