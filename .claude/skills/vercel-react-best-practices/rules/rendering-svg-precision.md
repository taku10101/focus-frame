## SVG座標の精度を下げる

SVGの座標値を小数点以下1〜2桁に丸めます。デザインツールは視覚的に区別がつかないにもかかわらず、ファイルサイズとパース時間を増大させる6桁以上の座標値をエクスポートすることがあります。

**悪い例（過剰な精度）：**

```svg
<path d="M 10.123456 20.654321 L 30.987654 40.123456" />
```

**良い例（十分な精度）：**

```svg
<path d="M 10.1 20.7 L 31 40.1" />
```
