# 🎨 FocusFrame

**集中するほど、名画が見えてくる。**

ポモドーロタイマー × ドット絵リビール PWA

## 何これ？

25分集中するたびにドット絵のマスが1つ開放。著作権切れの名画が徐々に姿を現す。完成するまで何の絵かわからない。

## 特徴

- 🕐 ポモドーロタイマー（25分集中 / 5分休憩）
- 🖼️ 著作権切れ名画のドット絵リビール
- 🎯 作品名は完成まで非公開（作者・ジャンルで選択）
- 📐 マス数カスタム（5×5〜N×N）
- 📱 PWA対応（オフラインOK）
- 💰 完全無料・サーバーレス

## Tech Stack

Next.js / TypeScript / Tailwind CSS / Canvas API / IndexedDB / Vercel
